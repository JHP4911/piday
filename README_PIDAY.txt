You can very carefully edit the file 'wpa_supplicant.conf' in the files/ directory
to add or change Wifi network setttings.

There are instructions in that file.

You can log into your Pi - username pi 
                         - password piday

After logging in, switch to the directory for our application:
cd workspace/piday

then to show output on the monitor:
./display
Press CONTROL+C to stop

To print anything on the LCD screen:
./showtext 'something to put on the LCD screen'
